IDLE Isekai Chill — Lancement automatique (Windows)
====================================================

OPTION A — Double-clic
  Jouer-IDLE-Isekai-Chill.bat

OPTION B — Si Windows bloque le .bat
  Clic droit sur play-idle-isekai-chill.ps1
  > Executer avec PowerShell

PREMIER LANCEMENT
  - Connexion internet requise
  - Accepte UAC (admin) si demande : installe Git + Node.js automatiquement
  - Le jeu se telecharge depuis GitHub puis s'ouvre sur http://localhost:5173/

RELANCE
  - Meme double-clic : mise a jour auto + relance

DOSSIER DU JEU
  %USERPROFILE%\IDLE-Isekai-Chill

BRANCHE AVANCEE (optionnel)
  powershell -ExecutionPolicy Bypass -File play-idle-isekai-chill.ps1 -Branch "main"
