# Chantier Myrion — assets

Arborescence préparée pour le MVP 3 visuel. Les PNG ne sont pas obligatoires tant que `available: false` dans `src/data/myrionWorksiteVisuals.ts`.

Voir **docs/MYRION_WORKSITE_ASSET_PIPELINE.md** pour les tailles, noms et procédure de remplacement.

## Dossiers

| Dossier | Contenu futur |
|---------|----------------|
| `backgrounds/` | Fonds de biome (prairie, forêt, mine, marais, cristal, astral) |
| `spots/` | Illustrations de spot sur la scène |
| `icons/` | Icônes ressources |
| `ui/` | Overlays état (actif, verrouillé, supervision, slot assignation) |
| `decorations/` | Éléments décoratifs optionnels (MVP 3+) |

Les placeholders CSS/HTML sont actifs sans fichier ici.
