# Gacha cinéma — Event Disagrea

Frames servies au runtime pour le diaporama d'ouverture (`GachaOpening variant="disagrea"`).

Source canonique : `assets/Gacha/cinema/disagrea/` (manifest : `assets/Gacha/event/disagrea/manifest.json`)

Servi en runtime sous `/gacha/cinema/disagrea/*` via `repoGachaAssetsPlugin` (vite).

Regénérer : `npm run import:disagrea-gacha-cinema`

Vidéo optionnelle (plus tard) : `npm run build:disagrea-gacha-video`
